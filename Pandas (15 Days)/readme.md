## 💡 Core Pandas Skills في الـ Study Plan دي

| الـ Method | الاستخدام |
|---|---|
| `pd.DataFrame()` | إنشاء DataFrame |
| `df.shape` | معرفة حجم البيانات (rows × cols) |
| `df.head(n)` | عرض أول n صفوف |
| `df.loc[condition]` | تحديد صفوف بشرط |
| `df.dropna()` | حذف القيم المفقودة |
| `df.fillna(value)` | تعبئة القيم المفقودة |
| `df.rename(columns={})` | إعادة تسمية الأعمدة |
| `df.astype(type)` | تغيير نوع البيانات |
| `df.drop_duplicates()` | حذف الصفوف المكررة |
| `pd.concat([df1, df2])` | دمج DataFrames |
| `df.pivot()` | إعادة تشكيل البيانات |
