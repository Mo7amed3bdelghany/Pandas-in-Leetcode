## 💡 Key Pandas Concepts You'll Learn

```python
# Filtering
df[df['col'] > value]
df.loc[condition, columns]

# String Operations
df['col'].str.contains(r'pattern')
df['col'].str.upper()

# Grouping & Aggregation
df.groupby('col').agg({'col2': 'sum'})

# Sorting
df.sort_values(by='col', ascending=False)

# Handling Missing Data
df.dropna(subset=['col'])
df['col'].fillna(value)

# Reshaping
df.pivot(index='col1', columns='col2', values='col3')
df.melt(id_vars=['col'], value_vars=['col2', 'col3'])

# Merging
df1.merge(df2, how='left', on='key')
pd.concat([df1, df2])
```
